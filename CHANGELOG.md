# Changelog

All notable changes to this project will be documented in this file.

The format is inspired by [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2025-12-12

### Added

- Frontend TypeScript types hardened across the app (removed most `any` usages)
- Shared type definitions for pagination, modals, and domain models
- Initial test setup and improved type-checking in the monorepo
- Documentation updates for configuration and worker runtime

### Changed

- Refactored frontend state and API utilities for better type safety
- Improved developer onboarding docs in the root README

## [1.0.5] - 2024-XX-XX

### Added

- Public open-source release of Voltage
- Multi-instance video encoding API based on FFmpeg
- Runtime worker pipeline for download, analyze, process, upload, and finalize steps
- React + TypeScript dashboard for monitoring jobs, logs, notifications, and workers
- Config package for centralized, type-safe configuration loading and validation
